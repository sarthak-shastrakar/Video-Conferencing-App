<h1>Video Conferencing App</h1>
A real-time video conferencing application built with the MERN stack.
<br>
<h2>Technology Used</h2>
<b>Frontend</b>
<ul>
  <li>React</li>
  <li>WebRTC</li>
  <li>Redux (or Context API)</li>
  <li>Material-UI (or other UI framework)</li>
</ul>
<b>Backend</b>
<ul>
  <li>Node.js</li>
  <li>Express.js</li>
  <li>MongoDB</li>
  <li>Socket.IO</li>
</ul>
